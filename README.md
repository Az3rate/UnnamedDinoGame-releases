# Unnamed Dino Game - Client Downloads

Download the latest release from the [Releases page](https://github.com/Az3rate/UnnamedDinoGame-releases/releases).

## Windows

1. Download the **win64** zip from Releases
2. Extract the zip
3. Run **DinoTamer.exe**

## Linux

1. Download the **linux64** zip from Releases
2. Extract the zip
3. Make executable: chmod +x DinoTamer_Linux.x86_64
4. Run: ./DinoTamer_Linux.x86_64

Linux requires Vulkan-capable GPU drivers (mesa-vulkan-drivers on Ubuntu/Fedora).
