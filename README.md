# Age of Beasts | Project: Dinosaur

Download the latest release from the [Releases page](https://github.com/Az3rate/AgeOfBeasts-releases/releases).

## Windows

1. Download the **win64** zip from Releases
2. Extract the zip
3. Run **AgeOfBeasts.exe**

## Linux

1. Download the **linux64** zip from Releases
2. Extract the zip
3. Make executable: `chmod +x AgeOfBeasts-Linux`
4. Run: `./AgeOfBeasts-Linux`

Linux requires Vulkan-capable GPU drivers (`mesa-vulkan-drivers` on Ubuntu/Fedora/Mint).

## Server

The official dedicated server runs at `140.82.47.39:27015`. Connect from the in-game menu.

## Requirements

- **Windows**: DirectX 12 compatible GPU
- **Linux**: Vulkan-capable GPU with drivers installed
- **Network**: UDP port 27015 outbound
